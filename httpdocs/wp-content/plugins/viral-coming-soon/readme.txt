== Installation ==

The plugin is easy and fast to install and configure.

Via the Wordpress Dashboard:

1. Login to your Wordpress Dashboard
2. Go to "Plugins"
3. Click on "Add new"
4. Search for "Viral Coming Soon Page by Growtheme"
5. Click on "Install now"
6. Activate the Plugin

Uploading it manually

1. Download the Plugin from this page
2. Login to your Wordpress Dashboard
3. Go to "Plugins"
4. Click on "Add new"
5. Click on "Upload Plugin"
6. Select the plugin .zip archive from your computer
7. Upload the Plugin
8. Activate the Plugin

Using FTP

1. Download the Plugin from this page
2. Extract the plugin .zip archives to your computer
3. Open your FTP Programm
4. Connect to your FTP Server
5. Navigate to the "wp-content/plugins/" Folder
6. Upload the extracted Plugin Folder into the "wp-content/plugins" directory
7. Login to your Wordpress Dashboard
8. Go to "Plugins"
9. Activate the Plugin

== Frequently Asked Questions ==

= How do I get the emails of my subscribers? =

As a security mechanism, no subscriber data is stored on your web server. In order to save your subscribed data you have to sign up with a specialized Email Service Provider like AWeber or Mailchimp. Mailchimp offers a free plan for up to 2,000 subscribers.

= Why is this Plugin free? =

This plugin comes for free for users of my free email course on growing your Blog Audience. [Click here to join the free email course](http://driveblogtraffic.com/ "Grow your Blog Audience") so you don't miss out on the valuable free lessons.

= I've disabled the plugin but it still shows =

Please check that you have cleared the cache of your browser and of any wordpress caching plugin you might use.
